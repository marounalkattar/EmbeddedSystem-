Embedded system mini project:

This is an Embedded system project at University La Sagesse. The hardware used in this project is shown in the photo.

Component used are:

1- Arduino mega 2560 x1
2- Pull up button x4
3- Potentiometer x3
4- Joystick x1
5- Lcd (16,2) x 1
6- RGB x1
7- LED BAR (10 LED) x1
8- Seven segment X2

About this project:

Embedded testing is the process of verification and validation of both software and hardware. It insures the defect free whole system including software and hardware. It is basically performed on hardware to find the defects. So, the purpose of this project is to test the board before using it. Actually, we are using a board from the laboratory of La Sagesse University-Polytech.

Some explanation of this project

This project is divided into 10 stages using state machine. In each state the seven segments will display a number (ex: for state 0 the seven segments will display 00). Once the LED will go on from the 10 led bar, the seven segments and the led bar use object oriented programming to set the pin and the number (classes). A color will be tested in the RGB (state 0 red, state 1 green, state 3 blue), and an input will be tested also. For more details, watch the attached video.
This project is tested in proteus before using on the board.

À propos de ce projet :

Les tests intégrés sont le processus de vérification et de validation des logiciels et du matériel. Il assure l'ensemble du système sans défaut, y compris les logiciels et le matériel. Il est essentiellement effectué sur le matériel pour trouver les défauts. Ainsi, le but de ce projet est de tester la carte électronique avant de l'utiliser.

Quelques explications sur ce projet

Ce projet est divisé en 10 étapes utilisant la machine d'état. Dans chaque état les sept segments afficheront un nombre (ex : pour l'état 0 les sept segments afficheront 00). Une fois que la LED s'allumera à partir de la barre de 10 leds, les sept segments et la barre de leds utilisent la programmation orientée objet pour définir la broche et le nombre (classes). Une couleur sera testée dans le RGB (état 0 rouge, état 1 vert, état 3 bleu), et une entrée sera également testée. Pour plus de détails, regardez la vidéo ci-jointe.
Ce projet est testé dans proteus avant d'être utilisé sur le tableau.
